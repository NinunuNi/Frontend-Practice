# Frontend-Practice
โปรเจกต์เว็บไซต์แฟชั่นที่เน้นความทันสมัย (Minimal &amp; Luxury) รองรับการแสดงผลทั้งคอมพิวเตอร์และ โทรศัพท์ และมีลูกเล่น Interaction ที่นุ่มนวล


- คุณสมบัติเด่น (Key Features)
Responsive Design: รองรับการแสดงผลสมบูรณ์แบบทั้ง Desktop, Tablet และ Mobile

  Custom Dropdown & Floating Label: ระบบเลือกหัวข้อในแบบฟอร์มติดต่อที่มาพร้อม Floating Label ลอยขึ้นเมื่อมีข้อมูล

  Smooth FAQ Accordion: ส่วนถาม-ตอบ (FAQ) ที่มีการเคลื่อนไหว (Animation) ที่ไหลลื่น ไม่กระตุก

  Interactive Gallery: แกลเลอรีรวบรวมคอลเลกชันที่จัดวางสัดส่วนอย่างสวยงาม พร้อมตัวหนังสือกำกับที่ปรับขนาดอัตโนมัติตามหน้าจอ (Fluid Typography)


- เทคโนโลยีที่ใช้ (Tech Stack)
  
  HTML5: โครงสร้างเว็บที่เป็น Semantic HTML

  CSS3: การจัดเลย์เอาต์ด้วย Flexbox, Grid และการทำ Animation/Transition

  Vanilla JavaScript: การจัดการ Logic ของ Dropdown และ Accordion โดยไม่ใช้ Framework
